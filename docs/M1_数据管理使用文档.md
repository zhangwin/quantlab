## 模块简介

M1 数据管理模块是 QuantLab 交易系统的数据底座，负责：

1. **数据下载与更新** — 从多种数据源拉取 A 股日频行情，写入 Qlib 本地二进制文件
2. **时间隔离的数据访问** — 为所有下游模块提供严格遵守 `anchor_date` 的数据接口
3. **数据检视** — CSV 导出和 K 线交互图表

所有脚本位于 `quantlab/data/` 目录下，直接 `python` 运行即可。

```bash
# 1. 安装依赖
pip install -r quantlab/requirements.txt

# 2. 安装 Qlib
cd qlib && pip install -e . && cd ..
```

## 快速开始

以下 5 步从零开始把环境跑通：

```bash
# 进入脚本目录
cd quantlab/data

# 第一步：下载 A股数据（首次约 200~500MB）
python data_download.py

# 第二步：查看数据状态，确认下载成功
python data_status.py

# 第三步：增量更新到最新日期
python data_update.py --source baostock

# 第四步：导出茅台 2024 年数据为 CSV
python data_export.py --symbols SH600519 --start 2024-01-01 --end 2024-12-31

# 第五步：画一张 K 线图
python data_plot.py --symbol SH600519 --start 2024-01-01 --end 2024-06-30 --show
```

或者用一键脚本完成环境搭建 + 验证：

```bash
python quantlab/setup_and_verify.py
```

## 脚本详细说明

### data_download.py 下载数据

首次使用时运行，从 GitHub 下载 Qlib 预处理好的 A 股日频数据。

```bash
# 默认下载到 ~/.qlib/qlib_data/cn_data/
python data_download.py

# 指定目标目录
python data_download.py --target-dir /data/qlib/cn_data

# 下载美股数据
python data_download.py --region us

# 强制重新下载（覆盖已有数据）
python data_download.py --force
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--target-dir` | `~/.qlib/qlib_data/cn_data` | 数据存放目录 |
| `--region` | `cn` | 数据区域：`cn`（A股）/ `us`（美股） |
| `--force` | 否 | 强制重新下载 |

下载完成后数据目录结构：

```
cn_data/
├── calendars/day.txt            # 交易日历，每行一个日期
├── instruments/
│   ├── all.txt                  # 全部股票及起止日期
│   └── csi300.txt               # CSI300 成分股
└── features/{SYMBOL}/           # 每只股票一个目录
    ├── open.day.bin             # float32 二进制，按日历顺序
    ├── high.day.bin
    ├── low.day.bin
    ├── close.day.bin
    ├── volume.day.bin
    └── factor.day.bin           # 复权因子
```

### data_update.py 增量更新

每日收盘后运行，将本地数据更新到最新。支持三种数据源。

```bash
# 更新到今天（默认用 Yahoo Finance）
python data_update.py

# 更新到指定日期
python data_update.py --end-date 2025-03-10

# 用 baostock 更新（推荐，A股更稳定）
python data_update.py --source baostock

# 从本地 CSV 文件导入
python data_update.py --source csv
```

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--end-date` | 今天 | 更新到哪天，格式 `YYYY-MM-DD` |
| `--source` | `yahoo` | 数据源：`yahoo` / `baostock` / `csv` |
| `--data-dir` | `~/.qlib/qlib_data/cn_data` | Qlib 数据目录 |
| `--market` | `csi300` | 股票池 |

**三种数据源对比：**

| 数据源 | 安装 | 优点 | 缺点 |
|--------|------|------|------|
| `yahoo` | 无需额外安装 | Qlib 内置，自动更新指数成分 | A 股数据可能延迟/缺口 |
| `baostock` | `pip install baostock` | 免费无限速，A 股全覆盖 | 需额外安装 |
| `csv` | 无 | 完全可控，自定义数据源 | 需手动准备文件 |

**CSV 导入方式：** 将 CSV 文件放到 `~/.qlib/qlib_data/csv_import/`，文件名为股票代码（如 `SH600519.csv`），列格式：`date,open,high,low,close,volume,amount`。

### data_export.py 导出 CSV

将 Qlib 二进制数据导出为可读的 CSV 文件。

```bash
# 导出单只股票
python data_export.py --symbols SH600519 --start 2024-01-01 --end 2024-06-30

# 导出多只股票（逗号分隔）
python data_export.py --symbols SH600519,SH601318,SZ000001 --start 2024-01-01 --end 2024-06-30

# 指定输出目录
python data_export.py --symbols SH600519 --start 2024-01-01 --end 2024-06-30 --output ./my_data

# 合并多只股票为一个文件（增加 symbol 列区分）
python data_export.py --symbols SH600519,SH601318 --start 2024-01-01 --end 2024-06-30 --merge
```

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `--symbols` | 是 | — | 股票代码，逗号分隔 |
| `--start` | 是 | — | 起始日期 |
| `--end` | 是 | — | 截止日期 |
| `--output` | 否 | `./export` | 输出目录 |
| `--merge` | 否 | 否 | 合并为单个 CSV |

**股票代码格式：** 沪市 `SH` 开头（如 `SH600519`），深市 `SZ` 开头（如 `SZ000001`）。

输出 CSV 格式：

```csv
,open,high,low,close,volume
2024-01-02,1713.0000,1728.0000,1710.0100,1726.0000,15234567.0000
2024-01-03,1720.0000,1725.8800,1705.0000,1708.0000,13456789.0000
```

### data_plot.py 画 K 线图

生成带均线和成交量的交互式 K 线图，保存为 HTML 文件。A 股配色（红涨绿跌）。

```bash
# 基本用法（保存为 HTML）
python data_plot.py --symbol SH600519 --start 2024-01-01 --end 2024-06-30

# 自定义均线周期
python data_plot.py --symbol SH600519 --start 2024-01-01 --end 2024-06-30 --ma 5,20,60

# 指定输出文件名
python data_plot.py --symbol SH600519 --start 2024-01-01 --end 2024-06-30 --output maotai.html

# 保存并在浏览器打开
python data_plot.py --symbol SH600519 --start 2024-01-01 --end 2024-06-30 --show
```

| 参数 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `--symbol` | 是 | — | 股票代码 |
| `--start` | 是 | — | 起始日期 |
| `--end` | 是 | — | 截止日期 |
| `--ma` | 否 | `5,10,20` | 均线周期，逗号分隔 |
| `--output` | 否 | `{symbol}_{start}_{end}.html` | 输出文件路径 |
| `--show` | 否 | 否 | 同时在浏览器中打开 |

图表包含两个面板：
- 上面板：K 线蜡烛图 + 均线
- 下面板：成交量柱状图

## Python API 参考

开发 M2~M9 模块时，通过 Python API 调用 DataManager 和 DataViewer。

### DataManager

```python
from quantlab.data.data_manager import DataManager

dm = DataManager(
    provider_uri="~/.qlib/qlib_data/cn_data",
    market="csi300",          # csi300 / csi500 / all
    update_source="baostock", # yahoo / baostock / csv
)
dm.init_qlib()
```

**数据更新：**

```python
dm.get_latest_date()                    # → Timestamp('2025-03-07')
dm.ensure_data_updated("2025-03-10")    # → True（已更新）/ False（已最新）
```

**交易日历：**

```python
dm.get_trading_calendar("2024-01-01", "2024-12-31")  # → [Timestamp, ...]
dm.has_date("2024-01-01")  # → False（元旦休市）
dm.has_date("2024-01-02")  # → True
```

**行情数据（核心接口，时间隔离）：**

```python
# 获取截至 anchor_date 最近 N 个交易日的 OHLCV（N 是交易日，不是自然日）
# 返回 {symbol: DataFrame[open, high, low, close, volume]}
ohlcv = dm.get_ohlcv_before(anchor_date="2024-06-28", lookback_days=30)

# 截面数据（某天全部股票的单个字段）
close   = dm.get_close_prices("2024-06-28")   # Series[symbol → 收盘价]
opens   = dm.get_open_prices("2024-07-01")     # Series[symbol → 开盘价]
returns = dm.get_daily_returns("2024-06-28")   # Series[symbol → 日收益率]
```

**涨跌停价（基于前日收盘 ±10%）：**

```python
limit_up, limit_down = dm.get_limit_prices("2024-06-28")
```

**Alpha158 因子（供 M2 使用）：**

```python
features = dm.get_alpha158_features("2024-06-28")
# → MultiIndex DataFrame (instrument, datetime), 158 列因子
```

**行业分类 / 基准净值：**

```python
industry = dm.get_industry_map()               # Series[symbol → 行业名]
nav = dm.get_benchmark_nav("SH000300", "2024-01-01", "2024-12-31")
```

### DataViewer

```python
from quantlab.data.data_viewer import DataViewer

viewer = DataViewer(dm)

# CSV 导出
viewer.export_csv(["SH600519"], "2024-01-01", "2024-06-30", "./export")
viewer.export_portfolio_csv(["SH600519", "SH601318"], "2024-01-01", "2024-06-30", "./all.csv")

# K 线图
fig = viewer.plot_kline("SH600519", "2024-01-01", "2024-06-30")
viewer.show(fig)

# K 线 + Kronos 预测叠加（预测部分半透明）
fig = viewer.plot_kline_with_prediction("SH600519", hist_df, pred_df)

# K 线 + 买卖标记（红▲买入，绿▼卖出）
trades = [{"date": "2024-03-15", "symbol": "SH600519", "direction": "buy",
           "exec_price": 1700.0, "status": "filled"}]
fig = viewer.plot_kline_with_trades("SH600519", "2024-01-01", "2024-06-30", trades)

# 持仓概览（行业饼图 + 个股盈亏柱状图）
fig = viewer.plot_portfolio_overview(positions, current_prices, industry_map)
```

## 配置说明

回测配置文件 `quantlab/configs/backtest.yaml` 中与 M1 相关的字段：

```yaml
start_date: "2023-01-01"
end_date: "2025-03-01"
market: "csi300"
qlib_data_dir: "~/.qlib/qlib_data/cn_data"
```

`RollingWindow` 封装滚动窗口参数，供各模块共享：

```python
from quantlab.data.data_manager import RollingWindow

rw = RollingWindow()
rw.finetune_lookback      # 30  — Kronos 微调回看（交易日）
rw.predict_horizon         # 5   — 预测天数
rw.alpha_train_years       # 3   — Alpha158 训练窗口（年）
rw.alpha_retrain_interval  # 20  — 重训间隔（交易日）
rw.ic_lookback             # 60  — IC 加权回看窗口

# 自定义
rw = RollingWindow(finetune_lookback=60, backtest_start="2022-01-01")
```

## 常见问题

### Q: 数据下载很慢怎么办？

数据托管在 GitHub，国内可能较慢：
1. 配置代理后重试 `python data_download.py`
2. 改用 baostock 作为数据源：`python data_update.py --source baostock`
3. 手动下载后解压到 `~/.qlib/qlib_data/cn_data/`

### Q: 报错 "Calendar file not found"

本地数据不存在，先下载：

```bash
python data_download.py
```

### Q: 报错 "Yahoo update failed"

Yahoo 对 A 股不稳定，改用 baostock：

```bash
python data_update.py --source baostock
```

### Q: 行业分类显示 "unknown"

需手动准备 `~/.qlib/qlib_data/industry_map.csv`：

```csv
symbol,industry
SH600519,食品饮料
SH601318,非银金融
SZ000001,银行
```

### Q: 如何使用 CSI300 以外的股票池？

所有脚本都支持 `--market` 参数：

```bash
python data_status.py --market csi500    # 中证500
python data_status.py --market all       # 全 A股
python data_export.py --market all --symbols SZ300750 --start 2024-01-01 --end 2024-06-30
```

### Q: 下游模块如何正确获取数据？

所有模块必须通过 DataManager，禁止直接调用 `qlib.data.D`：

```python
# 正确 ✓
ohlcv = dm.get_ohlcv_before(anchor_date="2024-06-28", lookback_days=30)

# 错误 ✗ — 绕过时间隔离，会引入前视偏差
from qlib.data import D
D.features(...)
```
